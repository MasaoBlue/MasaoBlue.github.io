IIDX CSV Viewer (Beta)

Tweet

このツールは、IIDX公式サイト でダウンロードしたCSVファイルを見やすく表示するツールです。

This tool makes it easy to display the CSV that was downloaded onIIDX official website.

β版のため、現状はSPのデータにしか対応していません。

This tool is Beta version, so only for single play.

How to use

  1. はじめに、現在の「.csv」ファイルを選択してください。

    First, select your current 「.csv」 file.

  2. 過去データとの比較を行う場合、過去の「.csv」ファイルを選択してください。

    If you want to compare with old data, then select your old 「.csv」 file.

  3. ボタンをクリックするとスコアを表示します。

    Click to start!

  4. ご質問、ご要望はtwitterまでお気軽に。

    Please feel free to ask on twitter.
