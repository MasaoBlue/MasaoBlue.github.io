開発したものを公開していく場所です。

Tools

  • IIDX CSV Viewer
    IIDXの公式CSVを読み込んで見やすく表示するツールです。県別TOPとの比較表示もできます。
  • IIDX Top Rankers Viewer
    IIDXの公式ページで公開されているTopRankerの情報を一覧表示・比較するツールです。
  • IIDX Key Logger v03 (2017/06/27)
    IIDX専用コントローラの入力を見える化するツールです。二重階段を無限に自動生成して練習できる機能を追加。

Old Version

  • IIDX Key Logger v02 (2017/06/19)

    階段譜面を無限に自動生成して練習できる機能を実装。

  • IIDX Key Logger v01 (2017/06/15)
    お手本通りに押してタスクをクリアしていく版。
  • IIDX Key Logger v00 (2017/06/12)
    IIDXの専用コントローラの入力を見える化するツールです。
  • IIDX CSV Viewer Beta1 (～2019/01/09)
    IIDXの公式CSVを読み込んで一覧表示するツールの古い版です。
